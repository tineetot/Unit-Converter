# Metric/Imperial Unit Converter


## 📖 Project Overview
The goal of this project was to create a functional dashboard where users can input a single numerical value and instantly see its equivalent across three different measurement categories: Length, Volume, and Mass.

Building this helped me master:
* **DOM Interaction:** Using `document.getElementById` to capture user input and update text content dynamically.
* **Math Operations:** Implementing precise conversion formulas and using `.toFixed(3)` to ensure a clean, readable output for users.
* **Event Handling:** Utilizing `addEventListener` to trigger calculations specifically when the "Convert" button is clicked.

---

## ✨ Key Features
* **Instant Multi-Conversion:** Convert one number into six different units simultaneously.
* **Precision Logic:** All results are rounded to three decimal places for professional accuracy.
* **Responsive UI:** A centralized, high-contrast design using a vibrant purple header and dark-mode results section.
* **Metric & Imperial Support:** Covers Meters/Feet, Liters/Gallons, and Kilograms/Pounds.
* **Optimized for Vite:** Leverages modern build tools for a lightning-fast development experience.


---

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for the conversion cards and input header.
* **CSS3:** Custom aesthetic featuring the `#6943FF` Purple palette, Google Fonts (Verdana), and subtle box-shadows.
* **JavaScript:** Logic for calculations, template literals, and event listeners.

---

## 📂 File Structure
- `index.html` - The UI structure including the input field and conversion display cards.
- `index.css` - Custom styles, featuring the responsive container and "digital" dashboard look.
- `index.js` - The logic handling the math formulas and DOM updates.
